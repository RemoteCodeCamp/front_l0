

### Next.js安装
初始化：

```shell
npx create-next-app@latest
```

:::info
[<font style="color:rgb(23, 23, 23);">Node.js 18.18</font>](https://nodejs.org/)<font style="color:rgb(23, 23, 23);"> or later.</font>

:::

启动：

```json
{
  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
```
